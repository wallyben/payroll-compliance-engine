## Severity levels

- **CRITICAL** — Net/gross/totals or missing critical data
- **HIGH** — Deductions, min wage, duplicates, PRSI, auto-enrolment
- **MEDIUM** — Pension cap, rate cap, USC thresholds, BIK
- **LOW** — PRSI Class A threshold check

## Config file

Configuration is loaded from `core/rules/ie_config_2026.json`. Required keys (see config contract):

- `minimum_wage.hourly_rate` (statutory minimum wage)
- `usc.exemption_limit` (USC exemption / threshold)
- `prsi.valid_prsi_classes`
- `prsi.class_a.weekly_threshold_lower`
- `auto_enrolment.eligibility.minimum_contribution_rate`
- `auto_enrolment.eligibility.annual_earnings_min`
